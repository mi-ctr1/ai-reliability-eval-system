# AI Reliability Evaluation System

## Overview
This project is an in-progress system designed to collect, validate, and analyze real-world data using automation and machine learning. The goal is to evaluate data consistency and AI-generated outputs, with a focus on system reliability and data integrity.

## Current Status
***In Progress*** | Initial work is focused on:
- Setting up data collection using browser automation
- Designing data structures for storing and processing results
- Exploring basic machine learning approaches for output evaluation

### Planned Components
1. Data Collection (Automation)
- Use Playwright to extract structured data
- Perform initial validation checks on collected data
2. Data Processing
- Clean and structure collected datasets
- Store results for analysis
3. Evaluation & Analysis
- Apply lightweight machine learning (scikit-learn)
- Detect anomalies and inconsistencies in structured data

## Tech Stack
- Python
- JavaScript (Playwright)
- scikit-learn
- LLM APIs (planned integration)

## Purpose

This project is part of a broader focus on building systems that improve reliability, automate validation, and evaluate the behavior of data and AI-driven workflows.
